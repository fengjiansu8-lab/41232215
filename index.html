曝光入門｜光圈 / 快門 / ISO

曝光三要素（曝光三角）是攝影的基礎，決定了一張照片的明暗與質感：

光圈 (Aperture)

控制鏡頭進光量的孔徑（如 f/1.8, f/8）。數值越小，光圈越大，進光量越多，且背景越模糊（淺景深）。

前往光圈詳細頁面 →

快門速度 (Shutter Speed)

控制感光元件曝光的時間（如 1/1000s, 1s）。高速可凝結瞬間，低速則可拍出動感（如流水）。

前往快門詳細頁面 →

ISO 感光度

相機對光線的敏感程度。光線不足時可調高，但過高會產生噪點（雜訊），影響畫質清晰度。

核心操作：模式 / 對焦 / 測光 / 白平衡

把這幾個弄熟，你就能更穩定脫離 Auto。

模式：Auto / A(Av) / S(Tv) / M
  • A(Av)：你控光圈（景深/背景糊），相機配快門
  • S(Tv)：你控快門（不糊/運動），相機配光圈
  • M：你控全部（固定光線、想要每張一致）
新手建議：先用 A / S 把成功率拉高，再慢慢玩 M。
對焦：AF-S / AF-C / 單點對焦
  • 靜態：AF-S
  • 走動/運動：AF-C
  • 最穩：單點對焦 → 人像就對眼睛
測光：多區 / 中央偏重 / 點測光
  • 新手：先用多區測光（成功率高）
  • 逆光人像：中央偏重 + 曝光補償 +EV
  • 主體很小：點測光對主體測亮度
白平衡：偏黃/偏藍快速修法
  • 先用 Auto WB（最快）
  • 室內黃光：切鎢絲燈
  • 陰天偏藍：切陰天
  • 想更好救：拍 RAW 後製調整

鏡頭介紹｜焦段 / 用途 / 怎麼選

不同的焦段決定了你的視覺範圍。新手建議從標準變焦鏡頭開始練習。

廣角

16mm - 35mm

適合壯闊風景、建築與室內空間。

標準

35mm - 70mm

最接近人眼視角，適合人像與街拍。

望遠

70mm - 200mm+

拉近遠景，適合運動、野生動物。

定焦

Prime Lens

大光圈、高畫質，適合人像氛圍。

構圖技巧｜九宮格 / 引導線 / 留白

構圖是攝影者的語言，透過安排畫面中的元素，引導觀者的視線並傳達情感。

中心構圖

將主體精確地放置在畫面的正中央，是最直觀且具備強烈視覺衝擊力的構圖方式，能產生穩定感。

三分法、九宮格

將主體置於「井」字交點上，能讓畫面更平衡且更具動感，避免死板的感覺。

引導線構圖

利用道路、圍欄等線條將觀者的眼睛引向主體，創造強烈的深度感與延伸感。

攝影常見錯誤與解法

在練習過程中遇到困難是正常的，這裡整理了新手的常見疑問：

Q: 為什麼我的照片總是糊糊的？
可能是快門速度過慢導致晃動，或者對焦失敗。建議先檢查快門是否高於安全快門（1/焦距）。
Q: 照片顏色看起來很怪（太黃或太藍）？
這是白平衡（WB）的問題。室內黃光可切換至「鎢絲燈」模式；陰天若偏藍可切換至「陰天」模式。
Q: 天空一片死白，沒有細節？
這稱為「過曝」。嘗試調低曝光補償 (EV-)，或對準明亮處測光以保留細節。
